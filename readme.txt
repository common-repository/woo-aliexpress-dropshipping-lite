== Description ==

Woocommerce Aliexpress Dropshipping Lite, an extension by CedCommerce allows sellers to Import Products from Aliexpress to their Woocommerce Store.

* The plugin requires the email address verification.
*The plugin access the Aliexpress Api(https://portals.aliexpress.com) using CedCommerce App Key, so for restricting unauthorised API Call: Email verification is required.
*The email address is not shared to any other 3rd party service. 

This extension provides various key features:-

* It allows seller to Import Product with Keyword or Category
* Allows sellers to add markup prices for the products before importing
* Import only desired products
* Ability to import the entire collection of product for respective filters

= Main Features of Free Version  =

* Import only 50 Products
* Create Filters based on Keyword and Category
* Easy and simple configuration.
* Easy to use and install

= Main Features of Premium Version =

* Import unlimited product from Aliexpress
* Advance Search Filters
* Import Products by Url
* Easy and simple configuration.
* Easy to use and install.

You can get premium version from [here](https://dropshipping.cedcommerce.com/solutions/aliexpress/woocommerce/)

== Installation ==

= Automatic installation =

In the search field type " Woocommerce Aliexpress Dropshipping Lite" and click Search Plugins. Once you've found our  'Woocommerce Aliexpress Dropshipping Lite' you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading the plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
